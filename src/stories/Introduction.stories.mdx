# Welcome to Blaueis

Blaueis is a React component library, it is named after the northernmost glacier of the Alps which lies near Ramsau, in Bavaria.
It aims at being a solid starting point for UI projects of various scopes, making the early phases of development quick, and
cost-effective.

Blaueis helps you build applications by providing often needed UI components with good user-experience bundled
with icons ([**Feather**](https://feathericons.com) is included in Blaueis), transitions, and animations. Relying on the defaults set by Blaueis results
in stunning interfaces which feel alive.

We plan on adding further components (e.g. components for building Forms), as well as improving current ones with a focus on making Blaueis more customizable,
and improving the interoperability of components. Check out our GitHub Project for more information.
